# 🎬 Netflix SQL Project

## 📖 About
This project analyzes the **Netflix Titles** dataset using **PostgreSQL** to answer 20 real-world business questions. It demonstrates SQL skills by exploring trends, content distribution, ratings, genres, directors, actors, and release patterns.

## 🛠️ Tech Stack
- PostgreSQL
- SQL
- pgAdmin 4

## 📌 SQL Concepts Covered
- Filtering & Sorting (`WHERE`, `ORDER BY`)
- Aggregation (`GROUP BY`, `HAVING`)
- Aggregate Functions
- Window Functions
- Common Table Expressions (CTEs)
- String & Date Functions
- Data Cleaning
- Subqueries
